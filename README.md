# Sistema de Registro de Inventario

Programa en Python que permite registrar un producto en el inventario solicitando su nombre, precio unitario y cantidad, calcula el costo total y muestra los resultados en consola.

1. Asegurate de tener Python 3 instalado.
2. Abre una terminal en la carpeta del proyecto.
3. Ejecuta el programa:



## Ejemplo de uso


Ingrese el nombre del producto: Lapiz
Ingrese el precio unitario: 500
Ingrese la cantidad: 3
Producto: Lapiz | Precio: 500.0 | Cantidad: 3 | Total: 1500.0




## Funcionalidades

- Solicita **nombre** (string), **precio** (float) y **cantidad** (int) al usuario.
- Valida que precio y cantidad sean valores numericos; si no, muestra un mensaje de error y vuelve a pedirlos.
- Calcula el **costo total** como `precio x cantidad`.
- Muestra los resultados en consola con formato claro.
- Codigo estructurado con **comentarios** en cada bloque.
